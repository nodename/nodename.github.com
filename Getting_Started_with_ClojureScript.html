Getting Started with ClojureScript

Reference: ClojureScript: Up and Running, by Stuart Sierra and Luke VanderHart

Each of the examples can be built with leiningen, the standard build manager and all-around toolkit for Clojure and ClojureScript. Leiningen's analogue to ANT's build.xml is the project.clj file, which is written in Clojure syntax.

First example: Hello, ClojureScript!

In this example the ClojureScript code simply writes a bit of HTML into the document. The HTML file runs the compiled js in its body. We compile the js from the ClojureScript source by invoking Leiningen's cljsbuild plugin from the command line. Note the cljsbuild options specified in project.clj.

Second example: Hello, ClojureScript compiler!

In this example the ClojureScript code defines a function main that writes a bit of HTML into the document. The function is marked with the export metadata tag to protect its name "main" from being munged during compilation, so the function can be called from another script on the html page. The HTML file loads the compiled js in its head, and then invokes main in its body.

This time we have no cljsbuild options in project.clj and we invoke the ClojureScript compiler interactively, specifying options in the Clojure code we type at the repl. Not that we'd usually do that, but it demonstrates compiling ClojureScript from within a Clojure program.

Third example: Hello, SVG!

Here we return to writing explicit HTML into the document to draw a circle using SVG. Once again we configure the build in project.clj, so we can compile the ClojureScript to js from the command line using lein cljsbuild.

Fourth example: Hello, Raphael!

Now let's draw the circle using Raphael.js. Our program calls several Raphael functions by name, so we want those names to be protected from munging. We specify those in the raphael-externs.js file.
